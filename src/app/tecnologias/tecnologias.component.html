PHP fue el primer lenguaje de programación que usé para de desarrollo web. Es un lenguaje que ha madurado bastante con el paso del tiempo, llegando a ser muy robusto en donde se puede hacer muchos tipos de implementaciones.
Yii, por su parte, es un framework (construido en php) basado en el patrón de diseño MVC con el cual he hecho mi primer aplicación web funcional de manera profesional. Siendo un framework robusto, seguro y fácil de usar.
JavaScript es un lenguaje que suelo usarlo para funciones concretas que manejo en el DOM, que lo complemento con una librería que al día de hoy se sigue usando: jQuery.
CSS he usado muy poco debido a que me apoyo de Fraweworks de diseño como Boostrap, o en este caso, Materialize. Sin embargo, conozco las cosas elementales.
Tengo experiencia en diseño y consultas de bases de datos SQL. Los dos gestores que más he usado son MariaDB y MySql, los cuales tienen el mismo nucleo.

Laravel es un framework MVC robusto para crear aplicaciones complejas y compatible con librerias de JS modernas; aunque no tengo mucha experiencia en el desarrollo de aplicaciones con esta herramienta.
Django es un framework construido en Python, ligero y con una sintaxis simple. Lo poco que he codificado aquí, se me ha hecho sencillo de comprender.
Angular es un framewok de JavaScript moderno que facilita mucho el control del DOM con poco código. Este sitio web está construído en angular.
Con Ionic es posible crear apps móviles híbridas y multiplataformas. Su integración nativa con ángular permite el fácil manejo de diversas librerías.
